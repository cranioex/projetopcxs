Componentes Identificados

Componentes da Placa F211

  • TIL113 (8x) — Optoacopladores Darlington (6 pinos)
  • SCL4068BE — NAND 8 entradas (todos HIGH = saída LOW)
  • CI adicional — Latch/Buffer (DIP-16)
  • Resistores — Limitação corrente LEDs (já na placa)

Componentes EXTERNOS (comprar/montar)

  • Resistores 1K (8x) — Para limitar corrente dos LEDs (~5mA)
  • Resistores 4K7 (8x) — Pullup para fototransistores
  • Fios jumper — ~20 fios macho-macho
  • Protoboard — Para montar os resistores

Alimentação

+5V Fonte bancada → VCC placa (SCL4068 + pullups)
USB PC → Arduino Mega (5V separado)
GND Comum entre fonte + Arduino + placa

Código: teste_f211_mega.ino
